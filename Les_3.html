🇳🇱 Cours simplifié : Accord de l’adjectief en néerlandais

📘 1. L’adjectief avant le nom (épithète)

Quand l’adjectief est placé devant le nom, on met généralement -e.

Mais il y a UNE exception : ❌ pas de -e si :

  • le nom est un het-woord
  • au singulier
  • avec een

Dans tous les autres cas, on ajoute -e ✅

📘 2. L’adjectief après le verbe (attribut)

Quand l’adjectief est après le verbe (être, sembler, rester…), on ne met jamais -e.

Exemple : De man is groot ❌

📘 3. Cas particuliers où on ajoute -e

  • Avec un possessif (mijn, jouw...) → ✅ mijn oude fiets
  • Avec un démonstratif (dit, dat...) → ✅ dit mooie boek
  • Quand l’adjectief est utilisé comme un nom → ✅ de arme
  • Au pluriel → toujours -e ✅ de mooie huizen
